# Spotify Clone Frontend

This is the React frontend for your Spotify clone.

## Run frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on `http://localhost:5173`.

## Backend requirement

Run your backend server separately in the `Backend` folder on `http://localhost:3000`.

## Connected APIs

- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/music`
- `GET /api/music/albums`
- `GET /api/music/albums/:albumId`
- `POST /api/music/upload`
- `POST /api/music/createAlbum`
