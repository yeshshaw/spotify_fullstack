🎧 Spotify Fullstack Clone

A fullstack Spotify-inspired music platform built with Node.js, Express, MongoDB, and React (Vite).
Users can authenticate, manage albums, and stream music through a clean UI.

⸻

🚀 Features

* 🔐 User Authentication (JWT-based)
* 🎵 Create & Manage Music
* 💿 Album Management
* ☁️ File Upload Support (ImageKit / storage service)
* 📡 REST API Integration
* ⚡ Modern Frontend with React + Vite

⸻

🛠️ Tech Stack

Backend

* Node.js
* Express.js
* MongoDB
* JWT Authentication

Frontend

* React.js (Vite)
* Axios (API calls)
* CSS
